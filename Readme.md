## Basic Bash Commands

  - Create directory
  ```Shell
  mkdir <dirname>
  ```

  - Show current location
  ```Shell
  pwd
  ```

  - Change directory
  ```Shell
  cd
  cd <dirname>
  cd ..
  cd -
  ```
  
  - Show all the files in current directory
  ```Shell
  ls
  ls -l
  ls -plsha
  ```
